1. Download Putty (https://putty.org)
1. Download Xming 
1. Install putty & xming (default setup - admin privilegues needed) 
1. Startup putty
1. Set X11 settings
  * Enable X11 Forwarding
  * Path to C:\Program Files (x86)\Xming\Xming.exe
1. Use putty
